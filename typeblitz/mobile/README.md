# 📱 Mobile — TypeBlitz

> React Native app — em desenvolvimento futuro.

## Planejado

- Treino de digitação em teclado físico via Bluetooth
- Sincronização de ranking com o backend
- Notificações de meta diária
- Histórico compartilhado com o web app

## Stack prevista

- React Native 0.73+
- Expo
- React Navigation
- Compartilhamento de lógica com o `frontend/src/utils/`

## Como contribuir

Quando o módulo for iniciado, rode:

```bash
cd mobile
npx create-expo-app .
npm install
npx expo start
```
