# 🛡️ Homelab Security Testing Project

## 📌 Sobre o Projeto

Este projeto consiste em uma aplicação web desenvolvida inicialmente com o auxílio do Sonnet 4.6, utilizada como base para estudos práticos em segurança da informação.

O objetivo principal é utilizar essa aplicação em um ambiente controlado (servidor caseiro / homelab), onde realizo testes de segurança, análise de vulnerabilidades e implementação de correções diretamente no sistema.

Toda a parte de segurança — incluindo identificação de falhas, exploração e mitigação — é conduzida manualmente por mim como forma de aprendizado prático em cibersegurança.

---

## 🎯 Objetivos

* Criar um ambiente realista para testes de segurança
* Identificar vulnerabilidades comuns em aplicações web:

  * SQL Injection
  * XSS (Cross-Site Scripting)
  * CSRF (Cross-Site Request Forgery)
  * Falhas de autenticação/autorização
* Praticar técnicas de exploração de vulnerabilidades
* Implementar correções e boas práticas de segurança
* Evoluir continuamente o sistema com base nos testes realizados

---

## 🧠 Metodologia

A aplicação foi gerada inicialmente como base funcional. A partir disso, o fluxo de aprendizado segue:

1. Deploy da aplicação no servidor caseiro (homelab)
2. Análise de possíveis vulnerabilidades
3. Exploração controlada das falhas encontradas
4. Documentação dos problemas
5. Correção das vulnerabilidades
6. Revalidação da segurança após ajustes

---

## 🖥️ Ambiente de Testes

* Servidor local (homelab)
* Ambiente isolado para testes de segurança
* Simulação de cenários reais de ataque e defesa

## ⚠️ Aviso Legal

Este projeto é destinado **exclusivamente para fins educacionais**.

Todos os testes são realizados em ambiente controlado e sob minha responsabilidade.
**Não utilize técnicas aprendidas aqui em sistemas de terceiros sem autorização explícita.**
